# E-Ticaret Sepet Sistemi

## Proje Hakkında
Bu proje Yazılım Tasarım Örüntüleri dersi için geliştirilmiştir.
Kötü koddan başlayarak adım adım tasarım örüntüleri uygulanmıştır.

## Kullanılan Tasarım Örüntüleri
- **Faz 1 - Factory Pattern:** Ürün oluşturma
- **Faz 2 - Decorator Pattern:** Kargo ve hediye paketi ekleme
- **Faz 3 - Strategy Pattern:** İndirim ve ödeme yöntemi seçimi
- **Faz 3 - Observer Pattern:** Sepet bildirimleri

## Özellikler
- Kullanıcı kendi ürünlerini girer
- İndirim seçimi (%10, %20, Öğrenci)
- Ödeme yöntemi seçimi (Kredi Kartı, Havale, Taksit)
- Kargo ücreti otomatik eklenir
- Hediye paketi seçeneği
- Her işlemde bildirim gelir

## Branch Yapısı
- master → başlangıç
- faz1 → Factory Pattern
- faz2 → Decorator Pattern
- faz3 → Strategy + Observer Pattern
